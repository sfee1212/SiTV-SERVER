SiTV 代理服务器
#  🚀 项目简介
这是一个基于 Flask 的轻量级 TV 代理服务器，支持 PotPlayer 和 Web 播放。

---

## 🔥 一键部署
### 🖥️ Windows 用户 (只运行 EXE)

1. **一键安装**：
   打开 PowerShell，执行以下命令（将 URL 换成你的 Raw 链接）：
   ```powershell
   irm https://raw.githubusercontent.com/sfee1212/SiTV-SERVER/main/install_scripts/install.ps1 | iex
   ```
   *脚本会自动下载 EXE，安装到 `C:\SiTV`，并在桌面创建快捷方式。*
   
3. **或者手动运行**：
   直接把 `saileitv-server-win.1.0.exe` 发给用户，双击即可运行。

### 🐧 Linux 服务器 (一键部署)

1. **一键安装**：
   登录服务器，执行：
   ```bash
   curl -fsSL https://raw.githubusercontent.com/sfee1212/SiTV-SERVER/main/install_scripts/install.sh | sudo bash
   ```
   *脚本会自动下载二进制文件，安装到 `/opt/sitv`，并配置为 Systemd 系统服务自动运行。*
